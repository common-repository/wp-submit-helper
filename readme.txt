WP Submit Helper

== Description ==

WP Submit Helper es un plugin para Wordpress que ayudar a automatizar el envio de las nuevas entradas en un blog a sitios de Social Bookmarking.

Este plugin soporta el envio a sitios Pligg y a Simpy. Para poder enviar las entradas del blog sera necesario registrar primero una cuenta en cada uno de esos sistemas.

La idea surge basicamente como parte de mi trabajo de final de mis estudios de E.T. en Informatica de Gestio, dentro del marco de Technology Enhanced Learning.

Dentro del trabajo, entre otras actividades, tambien encontramos este otro plugin: WP Easy Biblio

La version actual es bastante sencilla pero marca las bases para un plugin mas potente, en el que como primera nueva funcionalidad se podria incluir la posibilidad de seleccionar si la entrada se debe enviar o no a los sitios de SB, seleccionable desde el propio editor del WP como una opcion mas.

== Installation ==
1. Upload the 'wp-submit-helper.php' file to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Under Settings, click 'Submit Helper' and modify your settings accordingly.
4. That's it!
